Inventory System Requirements

  1. Name: Manager Interface
    Type: Functional
    Summary: User interface accessed by managers. Allows adding/removing/modifying items in the database, create new orders
  2. Name: Employee Interface
    Type: Functional
    Summary: User interface accessed by employees. Allows viewing items, modifying stock, receiving shipments.
  3. Name: add_item
    Type: functional
    Summary: Items should be added to data base.
  4. Name: remove_item
    Type: functional
    Summary: Items should be removed to data base.
  5. Name: view_item
    Type: functional
    Summary: Item information should be displayed.
  6. Name: modify_item
    Type: functional
    Summary: Items members should be modifiable.
  7. Name: item_info
    Type: functional
    Summary: Item will contain the follow, Name,Price,Product_id,Product_Count,Item_Location
  8. Name: store_data_list
    Type: risk
    Summary: Must interface with Access
  9. Name: pull_total_items
    Type: functional
    Summary: Must pull total number of individual product
